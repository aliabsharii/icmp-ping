# ICMP Ping Manager

ابزاری ساده و سبک برای مدیریت پاسخ به Ping (ICMP) در سرورهای لینوکسی.

با این اسکریپت می‌توانید تنها با یک منوی تعاملی:

✅ پاسخ به Ping را غیرفعال کنید  
✅ پاسخ به Ping را فعال کنید  
✅ وضعیت فعلی Ping را مشاهده کنید  

این ابزار بدون نیاز به فایروال و به صورت مستقیم از طریق تنظیمات کرنل لینوکس کار می‌کند.
